What Are Claude Skills?

Claude Skills are pre-built instruction sets that extend Claude's capabilities for specific tasks. Think of them as "apps" for Claude Code - install a skill and Claude instantly knows how to handle specialized workflows like processing real estate leads, writing listing descriptions, or generating marketing content.

Skills are stored as simple markdown files that you add to your Claude Code configuration. Each skill contains detailed instructions, templates, and best practices for its specific domain.

# Install a skill with Claude Code
claude skill install bogen.ai/real-estate/fb-lead-processor
