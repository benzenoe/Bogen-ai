What Are Claude Skills?

Claude Skills are pre-built instruction sets that extend Claude's capabilities for specific tasks. Think of them as "apps" for Claude Code - install a skill and Claude instantly knows how to handle specialized workflows like processing leads, writing content, or generating reports.

Skills are stored as markdown files in your Claude Code configuration. Each skill contains detailed instructions, templates, and best practices for its specific domain.

# Add a skill to ~/.claude/skills/skill-name/SKILL.md
